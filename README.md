# Vulnerability Dashboard

Dashboard de gestión de vulnerabilidades hecho con React + Vite + TailwindCSS (shadcn/ui).

## Instalación

```bash
npm install
npm run dev
```
## Tecnologías
- React + TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- Lucide-react (iconos)
- Recharts (opcional, para gráficos)

## Requisitos
Asegúrate de tener instalado:
- Node.js v18 o superior
- npm v9+
